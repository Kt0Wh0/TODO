# TODO List — Fullstack Spring Boot Application 

Fullstack веб-приложение для управления проектами.  
Каждый проект содержит собственный список задач.  
Backend реализован на **Spring Boot**, frontend находится в этом же репозитории и раздаётся приложением.

## Возможности

###  Пользователи
- Регистрация и авторизация пользователей
- Работа только со своими проектами и задачами

### Проекты
- Создание проектов
- Просмотр списка проектов
- Редактирование проектов
- Удаление проектов

### Задачи
- Каждая задача принадлежит конкретному проекту
- Создание задач внутри проекта
- Просмотр списка задач проекта
- Удаление задач

### Интерфейс
- Веб-интерфейс на HTML / CSS / JavaScript
- Frontend и backend находятся в одном репозитории

## Технологический стек

### Backend
- Java 23
- Spring Boot 3.4.3
- Spring Web
- Spring Data JPA
- PostgreSQL
- Maven
- Lombok

### Frontend
- HTML
- CSS
- JavaScript (без frontend-фреймворков)

## Запуск проекта

### Подготовка базы данных

Используется PostgreSQL.

В базе данных должна существовать схема `todo`, а также все необходимые таблицы.
Hibernate не создаёт таблицы автоматически.
Настройки подключения находятся в application.properties

```java
spring.datasource.url=jdbc:postgresql://localhost:5432/postgres?currentSchema=todo
spring.datasource.username=your_name
spring.datasource.password=your_pass
```

## Запуск приложения
Через Maven (рекомендуется для разработки)
mvn spring-boot:run

После запуска приложение будет доступно по адресу:
http://localhost:8080

## Логика приложения
-Пользователь может создать несколько проектов
-Каждый проект содержит собственный набор задач
-Задачи не существуют вне проекта
-Проекты можно редактировать
-Удаление проекта приводит к удалению связанных с ним задач
